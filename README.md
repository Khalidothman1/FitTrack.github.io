## 🚀 How to Host on GitHub Pages

### Step 1 — Create a New Repository
- Go to [github.com](https://github.com) and create a new **public** repository
- Name it: `FitTrack.github.io`

### Step 2 — Upload the File
- Click **"Add file" → "Upload files"**
- Upload `index.html` to the root of the repo
- Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to **Settings → Pages**
2. Under **Source**, select **"Deploy from a branch"**
3. Branch: `main` | Folder: `/ (root)`
4. Click **Save**

### Step 4 — You're Live! 🎉
Your site will be available at:
```
https://khalidothman1.github.io/FitTrack.github.io/
```
> It may take 1–2 minutes to go live after saving.
